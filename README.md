## Gặp vấn đề với chuỗi GZIP Base64? Dán vào là xong.
- Dán chuỗi Base64 GZIP ở ô trái, kết quả JSON/text sạch ở ô phải.
- Không cài đặt, không tài khoản, chạy thẳng trên trình duyệt.
- Hỗ trợ cả trường hợp payload không phải JSON (hiển thị text thuần để bạn vẫn đọc được).

## 3 bước nhanh
1) Mở trang web.
2) Dán chuỗi GZIP Base64 vào ô trái (tool tự chạy).
3) Xem, tìm kiếm, và bấm **Sao chép** nếu cần.

## Tính năng nổi bật
- Giải nén & format tự động, báo trạng thái rõ ràng.
- Line number, gấp/mở khối JSON như trình chỉnh sửa code.
- Tìm kiếm kiểu VSCode: Ctrl/Cmd+F, Enter/Shift+Enter để đi tới/lùi kết quả.
- Sao chép một chạm, có fallback nếu trình duyệt chặn clipboard.
- Theme tối/sáng/tự động, hiệu ứng ripple; kéo thay đổi kích thước hai ô.
- Thông báo lỗi dễ hiểu: Base64 sai, GZIP hỏng, hoặc JSON lỗi.

## Lợi ích cho bạn
- Tiết kiệm thời gian: dán vào là thấy kết quả, không cài tool nặng.
- Đọc dễ hơn: tự nhận JSON, có line number và gấp/mở khối để đỡ rối mắt.
- Tra cứu nhanh: thanh tìm kiếm quen thuộc, đếm và nhảy giữa các match.
- Linh hoạt: chạy online nhẹ nhanh.
- Thân thiện: copy một chạm, đổi theme tùy mắt nhìn.

## Phím tắt
- `Ctrl/Cmd + F`: Mở/đóng thanh tìm kiếm.
- `Enter` / `Shift + Enter`: Tới / lùi kết quả tìm kiếm.
- `Esc`: Đóng thanh tìm kiếm.

## Câu hỏi nhanh
- **Có cần cài gì không?** Không. Mở trang là dùng.
- **Payload không phải JSON thì sao?** Tool sẽ hiển thị dạng text để bạn vẫn đọc được.
